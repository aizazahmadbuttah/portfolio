# Aizaz Ahmad Buttah - Portfolio

Welcome to my personal portfolio! I am a passionate Computer Science student specializing in Artificial Intelligence, Machine Learning, and Modern Web Development.

## Overview
This repository contains the source code for my professional portfolio website, designed to showcase my skills, projects, and experience. The site is fully responsive, interactive, and built with modern web technologies.

## Key Features
- **Fluid Animations & Transitions:** Built with Framer Motion for a smooth, app-like user experience.
- **Dynamic Project Filtering:** Easily explore my work across categories like Full-Stack Development and AI/ML Research.
- **Interactive Timeline:** A structured vertical timeline detailing my education and professional journey.
- **Premium Toast Notifications:** Built-in sleek communication feedback via react-hot-toast.
- **Print-Optimized CV:** A dedicated page `/cv` structured for downloading directly to PDF.

## Tech Stack
- **Frontend Framework:** React.js (Vite)
- **Styling:** Vanilla CSS, Tailwind CSS (for some projects)
- **Routing:** React Router DOM
- **Animations:** Framer Motion
- **Alerts/Toasts:** React Hot Toast

## Running Locally

To run this project locally, simply clone the repository and run:

```bash
# Install dependencies
npm install

# Start the development server
npm run dev
```
